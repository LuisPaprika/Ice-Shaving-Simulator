# 🎮 Ice Shaving Simulator

A simulation game where players fulfill customer orders within a time limit while managing ingredient supplies across each day.

---

## 🎮 Controls

- W / A / S / D – Move  
- Left Mouse Button (LMB) – Interact, Pickup, Place  

---

## 🧩 Gameplay

### Order System
Players must prepare orders based on customer requests accurately and within a limited time.

### Resource Management
Ingredients are limited each day, requiring players to manage supplies efficiently to avoid running out.

---

## 🍧 How to Prepare Orders

### Ice Shaving
1. Place an ice block into the shaving machine  
2. Put an empty cup under the machine  
3. Add syrup according to the customer’s order  

### 🍦 Ice Cream
1. Pick up a cone  
2. Interact with the ice cream fridge to scoop ice cream  

### 🧇 Waffle
1. Use a ladle to scoop batter  
2. Pour the batter into the waffle machine  
3. Activate the machine to cook  
4. Place the waffle on a plate  

Note: Orders may require one or two waffles (order visuals are work in progress)

---

## ▶️ How to Run

- Download build from Releases
- Run .exe

or

- Open project in Unity  
- Load scene: MainScene  
- Press Play
